Portfolio

  1. Gliss (Repo/Live Link)

    • A grocery management application that gathers recipes for you based on your pantry items.
    • Client-side build with React and deployed via Vercel. Server built with Node.js that connects to a PostgreSQL database that stores users, items, and recipes all deployed via Heroku. Utilizes JWT to manage authentication and and bcrypt for password hashing.
    • Tech:Client(React, React Router, Jest, Vercel, CSS3), Backend(Node.js, Express, REST, JWT, PostgreSQL, Knex.js, Mocha/Chai)

  2. Date Night Roulette (Repo/Live Link)

    • A web application that allows users to randomly pick a Netflix title and a restaurant in their area for a spontaneous date night.
    • Uses LocationiQ API to obtain users location param and returns lat/long coordinates, passes those to Zomato API and returns a random result in the area. Sends a genre type selected by the user to unogs API to get total results, second call then randomly selects start result, fetches info, then displays info to the user.
    • Tech: JavaScript, jQuery, HTML, CSS, APIs(LocationIQ, Zomato, unogs)

  3. Interactive Audio Quiz (Repo/Live Link)

    • A web application that quizzes the user on Audio knowledge, adds their score, delivers prompt on correct and incorrect answer, and prompts a retry at the end.
    • Uses jQuery to traverse and manipulate the DOM via event handlers, render, and generation functions, with all info maintained inside of the store object.
    • Tech: JavaScript, jQuery, HTML, CSS
